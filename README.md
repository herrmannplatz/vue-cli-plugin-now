## Getting started

Create or use a vue-cli 3.x project.

```bash
vue create my-vue-app
```

Install the vue-cli-plugin-now plugin

```bash
npm install --save-dev vue-cli-plugin-now
```

Invoke the plugin.

```bash
vue invoke vue-cli-plugin-now
```

Deploy your vue app.

```bash
now
```
